# Private Alpha Observation Guide

## What to observe
- Does support feel emotionally accurate, calm, and not overconfident?
- Are routed brain + pacing profile generally appropriate for the language used?
- Do low-confidence moments ask softer clarification instead of forcing interpretations?

## Emotional failure cases
- Cold or robotic reflection in grief, loneliness, shutdown, or conflict moments.
- Advice density that feels pressuring when user energy is low.
- Overly specific interpretation when user language is ambiguous.

## Routing mistakes
- Work/school/social routes selected with low confidence and high certainty wording.
- Safety fallback not calm enough when orchestration or memory fails.

## Pacing mistakes
- Abrupt jumps from reflection to task plans.
- More than 2-3 actionable steps when user shows fatigue.

## Support fatigue signals
- Repeated prompts/questions in one response.
- User feedback includes "too much" or "needed less advice".

## Robotic vs genuinely helpful
- Robotic: repetitive templates, rigid structure, no emotional calibration.
- Helpful: acknowledges uncertainty, offers optional next step, preserves agency.
